# Contributing

Contributions are welcome! Please open an issue or pull request on GitHub.

## Getting Started

1. Fork the repo
2. Create a feature branch: `git checkout -b feature/your-skill`
3. Follow the [Claude Agent Skills standard](https://www.anthropic.com/engineering/equipping-claude-with-agent-skills) for any new skill directories
4. Ensure `requirements.txt` is updated if you add dependencies
5. Open a PR against `main`
